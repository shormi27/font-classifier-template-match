# font-classifier-template-match


## Environment:

• Developed Using Python 3.7 and Anaconda on Windows 10
• IDE used PyCharm
• Pytesserct and Tesseract_OCR installations required.

Source\createDataForTemplates.py

Source\testDataGenerator.py
